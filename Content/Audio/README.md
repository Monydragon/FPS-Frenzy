# Audio

Release audio is compiled from selected CC0 OGG sources. Short cues live under `Sfx` and are built as `SoundEffect`; music and stingers live under `Music` and are built as `Song`.

Runtime IDs intentionally match the filenames without extensions. Source-to-local mappings and licenses are recorded in `../ThirdParty/asset-manifest.json` and `../../docs/asset-sources.md`.
